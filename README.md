
# 🏦 Mini Banking System

This project is a simple and interactive banking application developed using Python and Streamlit during my internship at YBI Foundation. The main objective of this project is to simulate basic banking operations with a secure and user-friendly interface.

The application allows users to create an account using personal details such as name, address, mobile number, and a 4-digit PIN. After successful account creation, users can securely log in using their name and PIN to access banking features.

The system includes functionalities like:
- Deposit Money
- Withdraw Money
- Check Account Balance
- View Transaction History
- Logout Feature

For better security, PIN verification is required for sensitive operations like withdrawal, balance checking, and transaction history viewing. The application also includes an account lock feature if a wrong PIN is entered more than three times.

All transactions are recorded with accurate date and time details to provide a realistic banking experience.
